Simple widget to show author's recent post, support RTL

== Description ==

###New in 1.6.1
* option to hide feature image
* option to hide author section

This is a simple widget
plugin, that allows you to choose one of your blog's authors (or a random pick),
and display its latest posts (as many posts as you'dd like)

this plugin does not include any desgin (css), this allow the widget to inherit the look of your theme
 - you may use the class of the widget to alter the desgin in your css file.

###Features
* Select single author or random author
* Select the number of recent post to display for the selected author
* select the post type you want to show
* show post date
* Display post excerpt
* you can select how many words to show in the excerpt per widget
* you may add additional link at the bottom of the widget for example: "View all Authors"
* translation ready with po file.

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Appearance > Widgets
4. Drop the widget on your side and continue!
